# Pokedex
 
Aplicativo de pokédex simples criado em React Native.

### Instalação

Caso apenas queira **utilizar** a pokedex, baixe o arquivo `Pokedex.apk` e instale em seu aparelho Android.

Para abrir o app em modo desenvolvedor:
- siga o setup passo-a-passo do react native [aqui](https://reactnative.dev/docs/environment-setup)
- clone o repositório
- na pasta do repositório, execute os seguintes comandos:
  - `npm install`,
  - `npx react-native run-android`
